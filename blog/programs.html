Programs I Use

I dual-boot Manjaro along with Windows 10, using Manjaro primarily. I prefer minimalistic software and dislike programs that are bloated or are slow to open.

Software

Web Browser
Mozilla Firefox
Mail Client
Mozilla Thunderbird
Mutt
Documents
LaTeX and Markdown with Pandoc. I have almost completely ditched Microsoft Word for LaTeX for all my undergrad works. For simpler tasks and general writing, I uses Markdown which is much more simpler to use than LaTeX. I also like the fact that I can open and edit a .tex and .md file in an editor quickly as compared to opening a .docx file and waiting for MS Word to start up. The real star of the show though is Pandoc, a universal document converter. Pandoc is hands down one of the best program I ever come across.
Presentation
LaTeX-beamer and Markdown with Pandoc. Similar to writing documents, LaTeX-beamer has completely replaced MS PowerPoint. With the power of Pandoc, Markdown files can also be converted to LaTeX-beamer presentation slides. Another intriguing tool is suckless sent, which I use for informal presentation slides. sent lets you write your slides in plaintext and encourages the Takahashi method of presentation.
Spreadsheets
SC-IM and pandas library for Python. You are probably expecting this already: I don't use MS Excel for spreadsheets. SC-IM is a terminal program and fulfils simple spreadsheet tasks and start up quickly. For larger datasets manipulation, I prefer to use the pandas library for Python. I used to use R, but preferred Python as I am more familiar with it.
PDF Reader
Zathura on GNU/Linux and Sumatra PDF on Windows. I used to hate the .pdf format in the past as Adobe Acrobat Reader always take awhile to start up and the program feels bloated and slow. But now I know better and chose my own readers. Zathura and Sumatra PDF are lightweight and quick to open, as someone who is required to read a lot of .pdf files, this is essential. They can open other document formats too, such as DjVu and ePub.
Text Editor/Programming
neovim and VS Codium. neovim fulfils most of my text and program editing tasks being quick to start up and highly configurable. Sometimes I use VS Codium for larger projects. VS Codium is the open source version of Microsoft's Visual Studio Code. It has myriads of extensions, build-in Git source control, debugger and IntelliSense, yet being lightweight and quick to start up at the same time. VS Code is also the most popular development environment according to the 2018 Stack Overflow survey.
Office Suite
LibreOffice. Although I don't use MS Word, MS PowerPoint, and MS Excel any more, people send them to me all the time. Thankfully, LibreOffice provides a free and open source alternative to MS Office. I don't use it as much, but has to when I really need to open a .docx or .pptx file.
